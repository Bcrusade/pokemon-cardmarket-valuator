# Code of Conduct

## Our Pledge

We want this project to be a welcoming and constructive place for contributors.

## Expected Behavior

- Be respectful
- Focus on facts and implementation details
- Accept constructive feedback
- Keep discussions technical and useful

## Unacceptable Behavior

- Harassment or personal attacks
- Discriminatory language
- Bad-faith contributions
- Deliberate disruption of project collaboration

## Enforcement

Project maintainers may remove comments, issues, or contributions that violate these rules.
